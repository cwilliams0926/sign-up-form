# sign-up-form
A simple sign-up form for Odin Project.

# How it's made 
**Tech Used**: HTML and CSS

Started by creating the layout in the HTML file. There are 4 main sections: the branding panel, the info section, the form card itself, and the submission section for the form. The branding panel was given an aside since it is placed to the left side of everything else. Everything else was placed inside a main tag. The form itself features some basic login info like first name, email, and password. Everything was layed out using flexbox. Some basic client-side validation is used for the emails and password. All fields are required.

This project put some of the knowledge I've learned about forms to use, but I think the biggest chunk of the project was properly laying it out with HTML and CSS. I think it just put a lot of what I've learned over the past few weeks into practice.
